# Smart Expense Tracker

A modern, offline, professional-grade desktop application built with Python and CustomTkinter. Designed to meticulously track daily expenses, visualize monthly spending trends dynamically via embedded Matplotlib schemas, and rigorously enforce granular budgetary limits.

## Highlights
- **Stunning UI**: Fully responsive 16:10 aspect ratio layout styled with `customtkinter`. Includes vibrant dark-mode aesthetics, Paytm-style interactive data-bubbles, and beautiful dynamic sidebars. 
- **Deep Analytics**: Tracks history automatically and embeds live **Pie Charts** & **Bar Graphs** natively using `FigureCanvasTkAgg`. 
- **Budget Engine**: Empowers the user to enforce global monthly limits alongside strict category-by-category boundaries. A live health dashboard monitors progress in real time.
- **Standalone Privacy**: Fully offline framework backed by highly secure Parameterised SQLite3 Data Management.

## Setup & Run
Make sure you have standard Python 3.x installed. Then, install dependencies via standard package manager:
```bash
pip install customtkinter matplotlib
```

To run the application:
```bash
python main.py
```
